# Personal Finance Dashboard 💰

Built an interactive finance dashboard to automate bank statement analysis with transaction categorization and spending visualizations.

## Features
- Upload bank statement CSV
- Auto transaction categorization
- Monthly spending analysis
- Pie charts and bar charts
- Expense trend visualization

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## Run Locally
1. Clone repo
2. Install dependencies
3. Run:
streamlit run app.py
